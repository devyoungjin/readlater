# Read Later

Instapaper로 나중에 읽을 글을 keep해두며 읽고 있었는데, tag에 대한 아쉬움 때문에 생성한 레포지토리.

읽고 싶은 글이 있으면, 이슈로 올리고 해당 글을 표현할 수 있는 tag는 label로 관리하여 나중에 관련 글들을 쉽게 찾고 읽을 수 있도록 하려고 한다.

그리고 다 읽은 컨텐츠는 요약하여 정리해볼 생각.
